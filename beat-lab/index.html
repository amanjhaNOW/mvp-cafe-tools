❓ Beat Lab Help

Controls:

• Space — Play/Stop

• Click cell — Toggle note on/off

• Shift+Click — Set accent (louder hit)

• Right-click — Cycle velocity (soft/medium/hard/accent)

• 1-8 — Switch patterns

• R — Randomize pattern

• C — Clear pattern

• M — Mute hovered channel

• S — Solo hovered channel

• ←/→ — BPM -/+ 1

• ↑/↓ — BPM -/+ 10

Features:

• 4 drum kits synthesized with Web Audio API

• 8 pattern slots with copy between them

• 12 preset beats (hip-hop, house, rock, etc.)

• Per-step velocity with visual indicators

• Swing control for groove

• Reverb, delay, and compressor effects

• Waveform visualizer

• Per-channel mute/solo/volume

• 8/16/32/64 step modes

• Save/load projects as JSON

• Fully local — no server needed
