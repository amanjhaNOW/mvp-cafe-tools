# 🎯 Job Pipeline — Track Your Hunt

**Day 2 Overnight Build** — March 25, 2026

A local-first Kanban board for tracking your job hunt, end to end.

## What It Does

- **6-stage Kanban board:** Wishlist → Applied → Phone Screen → Interview → Offer → Rejected
- **Drag-and-drop** cards between stages
- **Priority indicators** (high 🔥 / medium / low) with color-coded left borders
- **Stats dashboard** — total, applied, active, offers, rejected, response rate
- **Search** across company, role, and notes
- **CSV export** for backup or sharing with recruiters
- **JSON/CSV import** for bulk adding
- **Local storage** — your data stays on your machine, zero server dependency
- **Dark mode** UI (because it's 2026)
- **Keyboard shortcuts:** `N` to add, `Esc` to close, double-click to edit

## How to Use

```bash
open index.html
```

That's it. No npm, no build step, no server. Just open the file.

Comes pre-loaded with 4 sample jobs to show the layout. Edit/delete as needed.

## Fields Per Job

- Company, Role, Salary Range, Location, Work Mode
- Priority, Stage, Date Applied
- Job URL, Contact/Referral, Notes

## Tech

Single HTML file. Vanilla JS. localStorage. Zero dependencies.
